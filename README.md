# рҹ§© Example Problem

## Contains Duplicate

### Problem Statement

Given an integer array `nums`, return `true` if any value appears at least twice in the array.

### Java Solution

```java
class Solution {
    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> set = new HashSet<>();

        for (int num : nums) {
            if (set.contains(num)) {
                return true;
            }
            set.add(num);
        }

        return false;
    }
}
```

### Time Complexity

```bash
O(n)
```

### Space Complexity

```bash
O(n)
```
